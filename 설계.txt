드래그&드롭 빙고판 만들기

리소스 : 캐이름 입력 / 드래거블 요소 / 드로퍼블 요소 / 원래 상태로 되돌리기 / 실행 취소 / DB에 제출하기

캐이름 입력
- 입력하지 않으면 제출할 수 없다.

드래거블 요소 
- DB의 '보기' 에 있는 단어를 리스트로 불러와 드래거블 요소에 넣어 텍스트값을 갖게 한다.
- 마우스로 드래그하여 위치를 옮길 수 있다. 
- 드로퍼블 요소 위에 놓으면 해당 위치로 고정되며, 더이상 드래거블 가능하지 않다. 또한 이전 위치에 있던 드래거블 초기 위치는 까맣게 빈칸으로 남는다.
- 드로퍼블 이외의 요소 위에 놓으면 원래 초기 위치로 돌아간다.

드로퍼블 요소
- 4X4(16개)의 칸으로 구성된 빙고판의 각 칸은 드로퍼블 요소이다.
- 드래거블 요소가 없을때는 빈 상태, 드래거블 요소가 놓였을때는 찬 상태이다.
- 드래거블 요소가 놓이면 해당 빙고판에 드래거블 요소의 고유값이 할당된다.

원래 상태로 되돌리기
- 드로퍼블 요소를 모두 비운다.
(드래거블 요소가 하나 이상이라도 있는 경우, 드래거블 요소는 원래 위치로 돌아가며 할당값 및 빈/찬 상태를 초기화하고, 없는 경우에는 아무것도 하지 않는다.)

실행 취소
- 최근 드로퍼블박스에 추가된 드래거블의 상태를 이전으로 돌린다.
(즉, 드로퍼블 박스에 추가된 드래거블 정보를 저장해둬야함)

제출하기
- 캐이름 입력된 상태에서 드로퍼블박스가 모두 꽉 차면 제출이 가능하다. (어느 하나라도 빈 곳이 있다면 제출 불가능)
- 데이터베이스 '채점' 에 제출되고, 채점 답안에 따라 빙고 점수가 다르게 될 수 있다.

랜덤
- 보기에 주어진 18가지 드래거블 박스중 랜덤하게 16개를 골라 드로퍼블에 배치한다. (드래그하지 않아도 된다.)
- 다시 랜덤을 누를 경우, 재배치된다.

* 세션으로 확인
제출 버튼을 한 번 확인한 적이 있고, 시간대가 아니라면 빙고 게임에 참여할 수 없다.
(시간대가 안됐을 경우 드래거블 요소에 접근 불가능하도록 비활성화 시키기)

제출된 빙고판 채점하기
리소스 : 답안 입력 판 1개, 수정하기, 답 완성하기 버튼

수정하기 클릭 시 빙고칸에 EDITBOX 생성
각 단어로 채워넣을 수 있으며, 1글자 이상이어야 한다.
답 완성하기 버튼을 누르면 답안이 수정되어 DB가 새로 수정된다.

캐이름, 제출 판(단어), 빙고 수를 표로 볼 수 있으며, 빙고 수가 많은 사람부터 정렬이 가능하다.
